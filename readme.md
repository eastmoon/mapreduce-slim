# MapReduce slim project

此專案為一個以 C++ 語言開發的簡易 MapReduce 框架實作專案。

## 函式庫

此專案包括一下函式庫：

+ [Boost](https://www.boost.org/)

## 指令

+ 進入虛擬環境
```
dockerw start
```
> 以 Docker 啟動開發環境，結束請使用 "Ctrl + Z"

+ 執行內容
```
. run.sh
```
> 以此指令呼叫 c++ 相關編譯工具進行處理，並執行相關輸出結果
